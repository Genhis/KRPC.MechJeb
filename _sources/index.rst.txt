kRPC.MechJeb Documentation
==========================

Are you tired of manually giving commands to your rockets? Do you want to automate tourist transportation? Do you know basics of programming but don't want to bother with calculating precise steering yourself? Do you like to use MechJeb for controlling your rockets but want more automation? If all of this is true, then this KSP mod is just for you!

|introduction| With `a simple script <https://genhis.github.io/KRPC.MechJeb/tutorials/launch-into-orbit.html>`_, you can configure MechJeb autopilots and remotely control your rocket.

.. |introduction| raw:: html

   <strong>This addon to <a href="https://krpc.github.io/krpc">kRPC</a> provides remote procedures to interact with <a href="https://github.com/MuMech/MechJeb2">MechJeb 2</a>.</strong>
