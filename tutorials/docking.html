This script will dock any rocket with a targetted rocket. The script assumes the target is a ship and it is next to the target.

The complete program is available in a variety of languages:

Java, Python

First of all, we start with connecting to the server, initializing service instances and getting the active vessel.

        System.out.println("Starting the docking process");
        DockingAutopilot docking = mj.getDockingAutopilot();
        docking.setEnabled(true);

        Stream<Boolean> enabled = conn.addStream(docking, "getEnabled");
        enabled.setRate(1); //we don't need a high throughput rate, 1 second is more than enough
        synchronized(enabled.getCondition()) {
            while(enabled.get())
                enabled.waitForUpdate();
        }
        enabled.remove();

        System.out.println("Docking complete!");
        conn.close();
    }
